How To Link SASS with CSS:_

With SASS Already Installed and Using Visual Studio Code, In the Folder of Your Project Create the Standard "index.html" file and 2 New Folders:_ The "css" Folder with the Standard File "styles.css" and The Folder "sass" with the File "styles.scss". Then, Open the File "styles.scss" with Visual Studio Code and Open the Terminal Console Going to [Terminal]--->>[New Terminal] and Write the Next Commands(Without the quotation marks):_

C:\User\You\Desktop\YourProjectFolder> "sass --watch sass:css"

+-And then you Press ENTER. It will create a File "styles.css.map" in the Folder "css" and All ready! Now Every time you make a Change and Save in your File "styles.scss" it Will Automatically be Applied to your File "styles.css".
